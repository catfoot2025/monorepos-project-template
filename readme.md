## 初始化项目

```shell
pnpm init
```

注意！pnpm 不支持-y参数

执行完该命令后，在根目录下将创建一个package.json文件，内容如下：

```json
{
  "name": "monorepos-project-template",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": [],
  "author": "",
  "license": "ISC",
  "packageManager": "pnpm@10.13.0"
}

```
